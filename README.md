 💎 Diamond King

一款簡單的終端機小遊戲，玩家需扮演冒險者在地圖中收集寶石並躲避陷阱，在移動次數內成功取得所有寶石即可獲勝。

---

 🎮 遊戲玩法

- 玩家角色：`@`（紅色）
- 寶石：`*`（紫色）
- 靜態陷阱：`^`（淺藍色）
- 移動陷阱：`O`（淡紅色）
- 地圖：隨機生成，每場皆不同
- **勝利條件**：收集滿五顆寶石
- **失敗條件**：撞到陷阱或次數不足

---

## ⌨️ 操作方式

| 鍵位 | 功能         |
|------|--------------|
| `W`  | 上移         |
| `A`  | 左移         |
| `S`  | 下移         |
| `D`  | 右移         |
| `Q`  | 離開遊戲     |
| `R`  | 重新開始遊戲 |
